## ✨ Overview

Advance Quiz Bot Simulator is a powerful Telegram-inspired Quiz Hosting System built entirely inside a single HTML file.

🎯 Create Quizzes  
🎮 Play Interactive MCQs  
🏆 Track Performance  
📊 View Statistics  
📄 Export HTML & PDF Reports

No backend. No database setup. No installation.

# 🚀 Getting Started

1. Download `index.html`
2. Open it in your browser
3. Run `/create`
4. Import questions
5. Save using `/done`
6. Launch with `/quiz QUIZ_ID`
7. Enjoy 🎉
